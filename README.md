# Portfolio Using Internal CSS

This project contains a simple portfolio webpage developed using **HTML** with **internal CSS styling**.

## About the Project
- Built using basic HTML structure
- Styling is done using internal `<style>` tags
- Focused on clean layout and readability

## Purpose
This project is submitted as part of the **Nest Digital evaluation process** to demonstrate
fundamental frontend development skills.

## Technologies Used
- HTML5
- Internal CSS
